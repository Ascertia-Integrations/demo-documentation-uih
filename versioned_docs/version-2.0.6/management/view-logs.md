# View Logs

**View Logs** allows you to view all or specific service logs within in the deployment.

First select the service you wish to view the logs for.

<figure><img src={gitbookAsset1} alt="" /><figcaption></figcaption></figure>

The system will then start to show the relevant log.&#x20;

<figure><img src={gitbookAsset2} alt="" /><figcaption></figcaption></figure>

Should the user wish to stop viewing the logs, simply press `Ctrl+C` to stop.
