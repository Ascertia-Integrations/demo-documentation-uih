# Release Notes

## Version: 2.0.5 - June 2025

| Compatibility          | Version |
| ---------------------- | ------- |
| Unity Integrations Hub | 2.0.5   |

Introducing a new version of the Unity Portal API, redesigned to fully support all SigningHub integrations. This new release ensures complete compatibility with the latest SigningHub features, providing a seamless and enhanced integration experience.

**What’s new:**

* **Enhanced SigningHub Integration** – Full compatibility with the latest SigningHub features.
* **Improved Performance** – Optimized API response times for faster interactions.
* **Expanded Endpoints** – New and updated endpoints to support additional functionalities.
* **Stronger Security** – Enhanced authentication and data protection measures.
* **Simplified Integration** – Streamlined processes for easier adoption by developers
