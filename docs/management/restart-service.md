# Restart Service

**Restart Service** option restarts the running services and is use to apply changes that have been made to configuration files.

First select the service you wish to restart

<figure><img src="/img/gitbook/Screenshot from 2025-06-02 14-55-47.png" alt="" /><figcaption></figcaption></figure>

The service will now be stopped, started and the needed tools installed on the container and health checks performed.&#x20;

**Once completed the success message will be displayed.**
