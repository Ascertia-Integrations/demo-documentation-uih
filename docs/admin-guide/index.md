# Admin Guide

### Who is this guide for?

**This guide is for system administrators with deep software deployment knowledge.** 

This documentation details the deployment of a backend component that will act as a coordinator for business applications (integrations) deployed on other platforms (such as SharePoint and Microsoft Teams).

:::info
For Ascertia SaaS users such as SigningHub.com, the Unity Integrations Hub is already deployed and managed by Ascertia.
:::
